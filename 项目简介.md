# 项目简介

| 字段 | 内容 |
| --- | --- |
| 项目名称 | 《智能求职投递助手》 |
| 立项人 | 权欣旖 |
| 项目背景 | 求职投递过程中，候选人需要针对不同 JD 反复调整邮件标题和正文，并重复填写 HR 邮箱、公司、岗位、简历附件等信息；人工处理耗时、容易遗漏岗位关键词，也难以稳定体现个人背景与岗位需求的匹配度。 |
| 项目价值 | 通过 LLM 自动分析 JD 与用户预设背景，生成高针对性的投递标题和“总-分-总”邮件正文，并通过 SMTP 一键发送简历附件，提升投递效率、邮件个性化程度和岗位匹配表达质量。 |
| 作品类型 | 线上应用/软件 |
| 开发工具及模型 | Codex、Python、HTML/CSS/JavaScript、Claude Sonnet 4.6、OpenAI-compatible API、QQ SMTP |
| 产品原型截图（至少 3 张） | 原型截图 1：投递信息录入界面<br>原型截图 2：整段粘贴自动拆分与简历拖拽上传界面<br>原型截图 3：AI 生成邮件预览与 SMTP 发送确认界面 |
